# Tenet Plugin

**Gerador de conteúdo autônomo e inteligente com memória e integração visual.**

O Tenet é um plugin WordPress avançado projetado para automação de conteúdo editorial. Ele utiliza múltiplos provedores de Inteligência Artificial (OpenAI, Google Gemini, OpenRouter) para criar artigos completos, otimizados para SEO e com linkagem interna inteligente. Além disso, o plugin integra-se ao Pixabay para buscar e importar imagens destacadas automaticamente.

## Funcionalidades Principais

*   **Múltiplos Provedores de IA:** Suporte nativo para OpenAI (GPT-4), Google Gemini e OpenRouter.
*   **Integração Visual:** Busca e download automático de imagens via API do Pixabay, com sideload para a biblioteca de mídia do WordPress.
*   **Módulo de Memória e Linkagem:** Analisa posts anteriores para evitar repetição de conteúdo e insere links internos contextuais automaticamente.
*   **Geração Assíncrona:** Processamento em background para evitar timeouts do servidor durante a geração de artigos longos.
*   **Gestão de Conteúdo:**
    *   Painel dedicado para solicitação de novos artigos.
    *   Definição de Tom de Voz e Público Alvo.
    *   Lista de "Últimos Posts Gerados" com acesso rápido para edição, independente do status (Rascunho ou Publicado).

## Instalação

1.  Faça o upload da pasta `tenet` para o diretório `/wp-content/plugins/` do seu WordPress.
2.  Ative o plugin através do menu 'Plugins' no WordPress.
3.  Vá para **Tenet > Configurações** e insira suas chaves de API (OpenAI/Gemini e Pixabay).

## Requisitos

*   WordPress 5.0 ou superior.
*   PHP 7.4 ou superior.
